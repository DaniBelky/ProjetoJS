Para o seu agronegócio:

    Sistema fotovoltaico off grid

    Não é conectado à rede de distribuição. No lugar disso, ele faz uso de baterias. A energia excedente gerada pelos painéis solares é armazenada para poder ser utilizada em outro momento (como à noite, quando não há luz solar). As baterias são caras, tornando o sistema off grid pouco acessível. Além disso, ele necessita de manutenção mais frequente. É recomendado apenas para lugares nos quais não há rede de distribuição local.


    Preço: R$ 100,00
